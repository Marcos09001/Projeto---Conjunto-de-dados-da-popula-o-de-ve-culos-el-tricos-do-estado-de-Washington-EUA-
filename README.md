# Projeto---Conjunto-de-dados-da-popula-o-de-ve-culos-el-tricos-do-estado-de-Washington-EUA-
Análise da população de veículos elétricos do estado de Washington (EUA), com ETL e EDA em Python e dashboards em Power BI, explorando padrões de adoção, características técnicas dos veículos e distribuição por região.
